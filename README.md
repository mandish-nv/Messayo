# Messayo
This project is an ambitious effort to design and implement a modern, secure, and engaging online messaging platform. Our goal is to provide a reliable service for real-time communication that prioritizes user experience and privacy.

## Key Features
* **Real-Time Messaging:** Facilitates instant exchange of text messages and multimedia content.

* **Social Connectivity:** Includes robust features for fostering social interaction:

- Friend Requests and connection management.

- Comprehensive User Profile Management to build and manage a digital presence.

* **User-Friendly Interface:** Designed to be intuitive and accessible across different devices, ensuring a seamless user experience.

## Tools and Softwares used
*	Platform: Windows
*	Designing Tool: Figma
*	Front end: React
*	Back end: Express js
*	Database: MongoDb
*	Communication protocol: Websocket
*	IDE: VS Code
*	API Development: Postman, Thunder client 
*	Hosting: Cloudfare
*	Deployment: Netlify
